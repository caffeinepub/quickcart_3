# Specification

## Summary
**Goal:** Build FreshCart, a full multi-page static grocery delivery website using HTML, CSS, Bootstrap 5, and vanilla JavaScript — no backend or frameworks.

**Planned changes:**
- Create a Home page with a hero banner (image carousel, headline, search bar, CTA), featured categories, top deals, popular products grid (8+ cards), app download banner, and footer
- Create an About Us page with mission statement, founding story, team member cards, core values, and milestones timeline
- Create a Contact Us page with a validated contact form, office details, and a map placeholder
- Create 35 additional pages: All Categories, Fruits & Vegetables, Dairy & Eggs, Bakery & Breads, Meat & Seafood, Beverages, Snacks & Munchies, Breakfast & Cereals, Organic Products, Frozen Foods, Personal Care, Household Essentials, Baby Care, Pet Care, Product Detail, Search Results, Shopping Cart, Checkout, Order Confirmation, Order Tracking, Order History, User Profile, Login, Register, Forgot Password, Deals & Offers, Blog/Tips & Recipes, Blog Post Detail, FAQ, Privacy Policy, Terms & Conditions, Refund & Return Policy, Delivery Information, Store Locator/Coverage Areas, and Careers
- Implement a sticky responsive navbar with FreshCart logo, category links, search bar, cart icon with item-count badge, and login/account button; hamburger menu on mobile
- Implement client-side JavaScript: add-to-cart with localStorage persistence and live badge update, cart quantity increment/decrement with total recalculation, real-time search filtering, inline form validation (Contact, Login, Register, Checkout), FAQ accordion, and hero image carousel
- Apply a cohesive green/white/orange design theme across all pages: product cards with soft shadows, rounded corners, product image, name, price, discount badge, and Add to Cart button; consistent typography and spacing
- Integrate FreshCart logo, hero banner background image, and a generic product placeholder image across all pages

**User-visible outcome:** Users can browse a fully styled, multi-page grocery delivery website, add products to a persistent cart, navigate 38 pages, use search and forms, and experience a consistent responsive layout on all screen sizes.
